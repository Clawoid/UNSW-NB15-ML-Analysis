# UNSW-NB15-ML-Analysis

This project performs data cleaning, exploratory data analysis, and machine learning on the UNSW-NB15 network traffic dataset using R.

## Dataset
The UNSW-NB15 dataset is not included in this repository due to size.
Please download it from the official source and place it locally before running the scripts.

## Tasks Performed
- Data cleaning and preprocessing
- Data analysis
- Data visualization using ggplot2
- Classification using Logistic Regression
- Classification using Random Forest

## Tools and Technologies
- R
- ggplot2
- Logistic Regression
- Random Forest

## How to Run
1. Download the UNSW-NB15 dataset
2. Update the dataset path in the R script if needed
3. Run the script in RStudio or an R environment
